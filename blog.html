Q1: What is the difference Among map, filter, forEach, find?

Ans: map() method returns a new array by applying a given function to each element of an existing array. while forEach() method execute a provided function once for each element in an array but it doesn't modify the main array nor return a new array. and filter() method create a new array with all elements that pass a condition by a given function. while find() method returns the value of the first element in the array that pass the given condition.

Q2: What is the difference Among Local Storage, Session Storage & Cookies?

Ans: LocalStorage Allow To store data that is stored locally on the client's computer & this stored data can be accessed even after closing the browser or after restarting the Device while SessionStorage Does The Exject Things As LocalStorage but SessionStorage stored data can't be accessed after closing the tab or browser.

Q3: What is Event Loop? & How How Event Loop Works?

Ans: The Event Loop is a mechanism in JavaScript that allows the language to handle asynchronous programming in a non-blocking way. It's a continuously running process that checks the message queue and executes the callback functions registered in response to different events.
